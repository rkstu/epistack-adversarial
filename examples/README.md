# examples

Case study source registries for the three demonstration cases.

## covid_origins/

The contested case. Six independent Bayesian analyses of the same evidence span 23 orders of magnitude. The Rootclaim debate ($100K bet, 15 hours, two PhD judges) provides structured adversarial source material.

`sources.yaml` lists the primary sources with metadata: Scott Alexander's analysis, both judge decisions, Weissman's Bayesian analysis, Rootclaim's response, and the debate video transcripts.

## black_holes/ (in progress)

The settled case. Whether CERN's LHC could create dangerous black holes. The scientific consensus is clear, so the system should produce high confidence. The interesting question is whether it correctly surfaces the weakest assumptions and connects to historical parallels (the 1946 LA-602 "will the bomb ignite the atmosphere" analysis).

## eggs/ (in progress)

The open-ended case. "Are eggs good to eat?" is deliberately vague. The system needs to handle ambiguity by identifying what sub-questions need answering before the main question becomes tractable, rather than forcing premature conclusions.
